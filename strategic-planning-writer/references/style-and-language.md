# Style and Language

Use formal central SOE Chinese planning style.

## Core Style

- 严谨客观：以事实、数据、政策、逻辑支撑判断。
- 庄重正式：符合央企战略规划、公文、领导审阅材料语体。
- 简洁有力：减少修饰性副词，避免口号堆砌。
- 执行导向：任务、机制、时序、责任、资源要清楚。
- 战略牵引：不仅写工作安排，还要写取舍、路径和能力建设。

## Preferred Verbs

构建、完善、健全、推进、推动、优化、提升、强化、统筹、聚焦、服务、支撑、保障、夯实、拓展、培育、打通、贯通、协同、压实、建立、形成。

## Standard Terms

- 能源安全新战略
- 新型能源体系
- 全国油气管网一张网
- 油气管网设施公平开放
- "X+1+X"油气市场体系
- 管输能力、管容、负荷率
- 储气调峰、应急保供、互联互通
- 省级管网融入
- 管道完整性管理
- 数字孪生管网、智能管道、统一调度
- 战略性新兴产业、第二增长曲线
- 氢基能源、CCUS、新型储能、可信数据空间

## Banned or Weak Expressions

Replace these unless they are supported by specific content:

| Weak expression | Better approach |
|---|---|
| 这块、那个、基本上 | 该领域、该事项, or delete |
| 非常、极其、巨大 | use data or remove |
| 毫无疑问、显而易见 | 分析显示、数据表明 |
| 大力推进 | specify project, scale, mechanism, timetable |
| 持续强化 | specify what is strengthened and how |
| 不断完善 | specify current gap and improvement target |
| 取得显著成效 | state concrete outcomes |
| 形成良好局面 | state specific operating state |

## Judgment Language

Use certainty levels:

- confirmed fact: "截至...，...已..."
- evidence-based trend: "综合...研判，...呈现..."
- forecast: "预计...，在基准情景下..."
- recommendation: "建议..."
- uncertainty: "仍存在...不确定性，需要..."

## Paragraph Density

For 20,000+ character reports:

- each major subsection should have at least 3-5 substantial paragraphs
- each paragraph should contain one clear claim
- after policy citation, add enterprise implication
- after data, explain meaning
- after problem, point to response

## Good Planning Sentence Patterns

- "从国家战略要求看，...对油气管网基础设施提出了更高要求。"
- "从行业改革趋势看，...正在推动管网企业由资产运营向平台服务、规则服务和能力服务延伸。"
- "从集团发展基础看，...既构成下一阶段发展的重要支撑，也暴露出...短板。"
- "下一阶段，应围绕...，以...为牵引，重点推进...。"
- "通过...，形成...能力，为...提供支撑。"
