# Domain Lexicon

Use these terms consistently in National PipeChina strategic planning reports.

## Policy and Strategy

| Standard term | Use |
|---|---|
| 能源安全新战略 | Guiding thought and macro policy context |
| 新型能源体系 | Energy transition and infrastructure positioning |
| 现代能源体系 | National strategy and long-term positioning |
| 全国油气管网一张网 | Network integration and unified operation |
| 油气管网设施公平开放 | Market service and reform chapters |
| "X+1+X"油气市场体系 | Oil and gas reform analysis |
| 全国统一大市场 | Market mechanism and platform service |
| 国企改革深化提升行动 | Governance and reform chapters |
| 碳达峰碳中和 | First appearance; later can use "双碳" |

## Pipeline and Gas Business

| Standard term | Notes |
|---|---|
| 管输能力/管容 | Use 亿方/年 for natural gas, 万吨/年 for liquids |
| 负荷率 | actual throughput/design capacity |
| 干线管道 | cross-regional or backbone pipeline |
| 支线管道 | branch line connecting demand nodes |
| 联络线 | interconnection between pipeline systems |
| 互联互通 | physical and operating connection |
| 储气调峰 | include underground storage, LNG, pipeline flexibility where relevant |
| LNG接收站 | distinguish receiving capacity and actual throughput |
| 省级管网融入 | reform and integration topic |
| 托运商 | shipper/customer using pipeline service |
| 管输费 | tariff; do not invent tariff values |
| 管道完整性管理 | safety and lifecycle management |
| 高后果区 | safety risk management |

## Digital and Intelligent

| Standard term | Notes |
|---|---|

## New Energy and Strategic Emerging Industries

| Standard term | Notes |
|---|---|
| 战略性新兴产业 | Use full name first, then "战新产业" |
| 第二增长曲线 | Use carefully; tie to staged evidence |
| 氢基能源 | hydrogen, ammonia, methanol and derivatives |
| 绿氢/蓝氢/灰氢 | define by production route |
| CCUS | carbon capture, utilization and storage |
| CO2管道 | future infrastructure opportunity; mark maturity and regulation uncertainty |
| 新型储能 | excluding pumped hydro unless specified |
| 压缩空气储能 | may connect to underground space/salt cavern resources |
| 地下空间利用 | storage, hydrogen, compressed air, CO2 sequestration |
| 碳市场/碳资产 | use with compliance caution |
| 量子通信 | prospective security scenario along pipeline fiber resources |

## Standard Units

| Item | Unit |
|---|---|
| Pipeline mileage | 公里 or 万公里 |
| Natural gas capacity | 亿方/年 |
| Crude/product oil capacity | 万吨/年 |
| Storage working gas | 亿方 |
| LNG receiving capacity | 万吨/年 |
| Investment | 亿元 |
| Pressure | MPa |
| Diameter | DN or mm |
| Power capacity | 万千瓦 |
| Carbon emissions | 万吨CO2 |

## First-Use Rule

For specialized terms, write full name first:

- 碳捕集、利用与封存（CCUS）
- 液化天然气（LNG）
- 管道完整性管理
- 战略性新兴产业（以下简称"战新产业"）
