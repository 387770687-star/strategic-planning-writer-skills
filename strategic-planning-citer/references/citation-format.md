# Citation Format

按来源类型提供标准化引用格式。每条引用必须包含足以让读者定位原文的信息。

## 格式总则

1. 内文采用行内归因，不设脚注（中国央企规划报告惯例）。
2. 表格/图表下方标注"资料来源"，可附口径说明。
3. 引用编号 `[C-章-序号]` 在正文中以紧凑形式标注，与台账对应。
4. 引用不打断阅读流，归因文字自然融入正文。

---

## 一、政策文件类

### 中央/国务院文件

**格式**：
```
根据[发文机关]《[文件全称]》（[文号]，[年份]），...
```

**示例**：
- 根据国务院《关于促进天然气协调稳定发展的若干意见》（国发〔2018〕31号），...
- 根据中共中央、国务院《关于完整准确全面贯彻新发展理念做好碳达峰碳中和工作的意见》（中发〔2021〕36号），...

**必填字段**：发文机关、文件全称、文号、发布年份
**可选字段**：具体条款编号、发布日期

### 部委规章/规范性文件

**格式**：
```
根据[部委名称]《[文件全称]》（[文号]，[年份]），...
```

**示例**：
- 根据国家发展改革委《天然气管道运输价格管理办法》（发改价格规〔2021〕818号），...
- 根据国家能源局《油气管网设施公平开放监管办法》（国能发监管〔2019〕56号），...

### 省级政策文件

**格式**：
```
根据[省份][发文机关]《[文件全称]》（[文号]，[年份]），...
```

**示例**：
- 根据四川省发展改革委《四川省"十四五"天然气发展规划》（川发改能源〔2022〕XX号），...

### 引用表

| 字段 | 要求 |
|---|---|
| 发文机关 | 全称，不得省略"中共中央""国务院""国家"等层级标识 |
| 文件全称 | 使用书名号，书名号内使用全称不加引号 |
| 文号 | 完整文号，含"国发""发改XX"等机关简称和年份、序号 |
| 年份 | 发布年份；引用过5年的政策需在台账中标注时效提醒 |

## 二、统计数据类

### 国家统计局数据

**格式**：
```
[发布机构][年份]年数据显示，[指标全称]达到[数值][单位]。
```

**示例**：
- 国家统计局2024年数据显示，全国天然气表观消费量达到4,250亿立方米。
- 据国家统计局《中国统计年鉴2024》，全国油气管道总里程达到XX万公里。

**必填字段**：发布机构、数据年份、指标全称、数值、单位、统计口径

### 行业主管部门数据

**格式**：
```
据[机构]《[出版物名称]》（[年份]），...
```

**示例**：
- 据国家能源局《中国天然气发展报告2024》，全国天然气产量达到2,500亿立方米。
- 据国家能源局油气司数据，2024年全国地下储气库工作气量达到XX亿立方米。

### 数据引用要求

| 要求 | 说明 |
|---|---|
| 数值必须含单位 | 不得出现"产能300"这种无单位的数字 |
| 标注统计口径 | 如"规模以上""全口径""含/不含XX" |
| 标注基准年 | 如"截至2025年末""2024年全年" |
| 增长率含基期 | "同比增长X%，上年同期为Y" |
| 预测数据标注假设 | "基于XX假设，预计2030年达到..." |

## 三、企业公开数据类

### 年报/ESG报告

**格式**：
```
据[企业名称][年份]年年报/[ESG报告/社会责任报告]，[事实陈述]。
```

**示例**：
- 据国家管网集团2024年年报，集团在役油气管道总里程达到XX万公里。
- 据国家管网集团2023年社会责任报告，全年完成管输量XX亿吨油当量。

### 企业公开讲话

**格式**：
```
[企业名称][职务][姓名]在[场合]（[时间]）表示，...
```

**示例**：
- 国家管网集团董事长在2024年全国能源工作会议上表示，...
- 据国家管网集团公开发布的XX信息，...

**注意**：
- 如无法确认姓名和原话，使用"据集团公开信息"或"据集团公开资料"
- 不要编造不存在的讲话内容

## 四、国际/行业报告类

### 国际组织报告

**格式**：
```
据[机构全称]《[报告全称]》（[年份]），...
```

**示例**：
- 据国际能源署（IEA）《World Energy Outlook 2024》，全球天然气需求预计...
- 据Energy Institute《Statistical Review of World Energy 2024》，...

### 行业研究机构报告

**格式**：
```
综合[机构]等研究，...
```

**示例**：
- 综合中国石油集团经济技术研究院、中国石化经济技术研究院等研判，...
- 据XX咨询机构研究，...

**注意**：
- 国际来源需提供中英文机构名称和报告名
- 咨询机构/券商报告需标注"第三方研究，不代表本报告立场"

## 五、用户提供材料类

### 内部数据和文件

**格式（正文内）**：
```
据集团内部[数据/文件]（截至[年份]年末），...
```

**格式（台账）**：
```
来源类型：用户提供
提供方：[部门/渠道]
数据类型：[项目清单/财务数据/KPI数据/规划稿]
提供日期：[日期]
状态：待用户确认口径
```

**保密处理**：
- 正文不出现涉密具体数字，使用区间或定性描述
- 台账标注"内部数据，已做脱敏处理"
- 数据口径标注"以集团最终核定口径为准"

---

## 六、表下图下出处标注

### 表格下方

```
资料来源：[来源1]、[来源2]、[来源3]。
数据截至：[年份]年[月]。
注：[口径说明，如"含在建项目""不含省内支线"等]。
```

### 图表下方

```
数据来源：[来源名称]，[年份]。
说明：[必要时的数据处理说明]。
```

## 七、特殊场景引用格式

### 多来源综合判断

当结论基于多个来源综合研判时：

```
综合[来源A][年份]、[来源B][年份]、[来源C][年份]等研判，[判断结论]。
```

### 引用转引

当无法获取一手来源，必须通过二手来源引用时：

```
据[二手来源]引述[一手来源]数据，...
```

台账中标注"转引，未核实一手来源"。

### 预发布/征求意见稿

```
据[文件名称]（征求意见稿，[年份]），...
```

台账中标注"征求意见稿，可能调整，以正式发布版本为准"。

## 格式检查清单

引用交付前逐条检查：
- [ ] 政策类引用含文号
- [ ] 数据类引用含单位和口径
- [ ] 所有引用含年份/日期
- [ ] 企业引用注明来源文件类型
- [ ] 表下图下含出处标注
- [ ] 无"据悉""据有关方面""据相关数据显示"等模糊来源
- [ ] 无失效或无法访问的链接
- [ ] 引用编号与台账一一对应
