# Citation Format（引用格式规范）

本文件定义战略规划报告中引用的标准化格式。核心要求：**正文【1】标注 + 文末参考文献汇总**。

---

## 格式总则

1. **正文内**：采用上标方括号编号 `【1】`，紧跟在被引用的事实、数据或论断之后，不打断阅读流。
2. **文末**：按编号顺序列出「参考文献」汇总表，每条包含编号、来源全称、发布年份、来源类型标注。
3. **编号规则**：从 【1】 开始，按正文首次出现顺序依次递增。同一来源多次引用时使用同一编号。
4. **来源必须可溯源**：读者根据文末汇总中的信息能够定位到原始资料。

---

## 一、正文内引用格式

### 基本格式

```
事实陈述【1】。根据XX政策要求，需采取YY措施【2】。国家统计局数据显示，2024年全国天然气消费量达到4,250亿立方米【3】。
```

### 同一来源多处引用

同一文件/数据在多处引用时，使用首次分配的同一编号：

```
第一处：国家管网集团管道总里程达到XX万公里【4】。
...
第二处：集团管道资产规模持续扩大【4】。
```

### 综合多个来源的判断

```
综合研判显示，天然气需求将持续增长【5】【6】【7】。
```

## 二、文末参考文献汇总格式

### 基本格式

在报告正文末尾，生成「参考文献」章节：

```
## 参考文献

【1】国家统计局，《中国统计年鉴2025》，2025年。来源：知识库
【2】国务院，《关于促进天然气协调稳定发展的若干意见》（国发〔2018〕31号），2018年。来源：知识库
【3】国家能源局，《中国天然气发展报告2025》，2025年。检索自：https://www.nea.gov.cn/2025-xx/xx.html
【4】国家管网集团，2025年年度报告。来源：用户提供
【5】国际能源署（IEA），《World Energy Outlook 2025》，2025年。检索自：https://www.iea.org/reports/world-energy-outlook-2025
```

### 不同类型来源的标注方式

| 来源类型 | 标注格式 | 示例 |
|---|---|---|
| 知识库已有资料 | `来源：知识库` | `【1】国务院，《关于促进天然气协调稳定发展的若干意见》（国发〔2018〕31号），2018年。来源：知识库` |
| 官方网站检索获取 | `检索自：<完整URL>` | `【3】国家能源局，《中国天然气发展报告2025》，2025年。检索自：https://www.nea.gov.cn/xxx.html` |
| 用户提供材料 | `来源：用户提供` | `【4】国家管网集团，2025年年度报告。来源：用户提供` |
| 知识库 + 官网交叉验证 | `来源：知识库；交叉验证：<URL>` | `【1】国家统计局数据，2025年。来源：知识库；交叉验证：https://www.stats.gov.cn/xxx` |

### WebSearch 检索来源的标题超链接格式

对于通过 WebSearch 检索到的资料，在文末汇总中使用 Markdown 超链接格式，方便读者点击溯源：

```
【3】国家能源局，《[中国天然气发展报告2025](https://www.nea.gov.cn/2025-xx/xx.html)》，2025年。检索自国家能源局官网
【5】国际能源署（IEA），《[World Energy Outlook 2025](https://www.iea.org/reports/world-energy-outlook-2025)》，2025年。
```

**规则**：
- 标题部分使用 Markdown 超链接语法 `[标题](URL)`
- 确保 URL 完整、可访问
- 知识库来源如有对应的原始网址，也应附加超链接

## 三、各来源类型的引用要素

### 政策文件类

**必填要素**：发文机关、文件全称（含书名号）、文号、发布年份

**汇总格式**：
```
【N】[发文机关]，《[文件全称]》（[文号]），[年份]年。来源：[来源类型标注]
```

**示例**：
```
【1】国务院，《关于促进天然气协调稳定发展的若干意见》（国发〔2018〕31号），2018年。来源：知识库
【2】国家发展改革委，《天然气管道运输价格管理办法》（发改价格规〔2021〕818号），2021年。检索自：https://www.ndrc.gov.cn/xxx
```

### 统计数据类

**必填要素**：发布机构、数据名称/出版物名称、数据年份、数值、单位、统计口径

**汇总格式**：
```
【N】[发布机构]，《[出版物/数据集名称]》，[年份]年。[指标]达到[数值][单位]。来源：[来源类型标注]
```

**示例**：
```
【3】国家统计局，《中国统计年鉴2025》，2025年。全国天然气表观消费量4,250亿立方米。来源：知识库
【4】国家能源局，《中国天然气发展报告2025》，2025年。全国天然气产量2,500亿立方米。检索自：https://www.nea.gov.cn/xxx
```

### 企业公开数据类

**汇总格式**：
```
【N】[企业名称]，[文件类型]，[年份]年。来源：[来源类型标注]
```

**示例**：
```
【5】国家管网集团，2025年年度报告。来源：用户提供
【6】国家管网集团，2025年社会责任报告。检索自：https://www.pipechina.com.cn/xxx
```

### 国际/行业报告类

**汇总格式**：
```
【N】[机构全称]，《[报告全称]》，[年份]年。检索自：[URL]
```

**示例**：
```
【7】国际能源署（IEA），《[World Energy Outlook 2025](https://www.iea.org/reports/weo2025)》，2025年。
```

## 四、表下图下出处标注

### 表格下方

```
资料来源：【3】【4】【5】。
数据截至：[年份]年[月]。
注：[口径说明，如"含在建项目""不含省内支线"等]。
```

### 图表下方

```
数据来源：【3】，[年份]年。
注：[必要时的数据处理说明]。
```

使用引用编号指向文末汇总，而非在图表下方重复完整引用信息。

## 五、特殊场景

### 多来源综合判断

正文：
```
综合研判，天然气需求将持续增长【3】【5】【7】。
```

文末汇总中 【3】【5】【7】 各自独立列出完整来源。

### 引用转引（无法获取一手来源时）

```
【N】[二手来源]引述[一手来源]数据，[年份]年。检索自：[URL]。注：转引，未核实一手来源。
```

### 预发布/征求意见稿

```
【N】[文件名称]（征求意见稿），[年份]年。来源：[来源类型]。注：征求意见稿，以正式发布版本为准。
```

---

## 六、格式检查清单

引用交付前逐条检查：
- [ ] 正文中每个事实性数据/政策引用后均有 【N】 标注
- [ ] 编号从 【1】 开始，按出现顺序递增，无跳号
- [ ] 文末参考文献汇总中的编号与正文一一对应
- [ ] 政策类引用含发文机关、文件全称、文号
- [ ] 数据类引用含单位、口径、基准年
- [ ] 每条引用末尾标注了来源类型（来源：知识库 / 检索自：URL / 来源：用户提供）
- [ ] WebSearch 检索来源附有标题超链接
- [ ] 无模糊来源（"据悉""据有关方面""据相关数据显示"等）
- [ ] 无失效或无法访问的链接
- [ ] 表下图下出处标注使用引用编号（而非重复完整信息）
