# Citation Ledger Template（文末参考文献汇总模板）

本模板定义报告文末「参考文献」汇总的标准格式。正文中的每个 【N】 标注在汇总中有且仅有一条对应记录。

---

## 编号体系

### 编号规则

从 【1】 开始，按正文首次出现的顺序依次递增。全文统一编号，不分章节。

同一来源在正文中多次引用时，使用同一编号（首次出现时分配的编号）。

### 编号示例

```
正文第1段：...管道总里程达到XX万公里【1】。
正文第2段：...根据国务院政策要求【2】，需加快储气库建设。
正文第3段：...天然气消费量持续增长【3】。
正文第3段：...管道资产规模扩大【1】。（← 同一来源再次引用，沿用【1】）
```

---

## 文末参考文献汇总模板

报告正文结束后，插入以下章节：

```
## 参考文献

【1】[来源全称]，[年份]年。[补充信息]。来源：知识库
【2】[发文机关]，《[文件全称]》（[文号]），[年份]年。来源：知识库
【3】[发布机构]，《[报告/数据名称]》，[年份]年。检索自：[完整URL]
【4】[来源全称]，[年份]年。来源：用户提供
【5】[机构名称]，《[报告名称]》，[年份]年。检索自：[完整URL]
...
```

### 每条记录的必填字段

| 字段 | 说明 | 是否必填 |
|---|---|---|
| 编号 | 【1】【2】【3】... | 是 |
| 来源全称 | 完整名称，政策类含书名号和文号 | 是 |
| 发布年份 | 文件/数据的发布年份或所属年份 | 是 |
| 来源类型标注 | 来源：知识库 / 检索自：URL / 来源：用户提供 | 是 |
| 补充信息 | 具体指标、数值、口径等（统计数据类） | 视来源类型 |

## 来源类型标注对照表

| 场景 | 文末标注 |
|---|---|
| 数据和来源从已上传知识库中获取 | `来源：知识库` |
| 通过 WebSearch 在官方网站检索获取 | `检索自：https://www.xxx.gov.cn/xxx` |
| 用户上传或提供的内部材料 | `来源：用户提供` |
| 知识库有该数据，但通过官网交叉验证了准确性 | `来源：知识库；交叉验证：https://www.xxx.gov.cn/xxx` |

## WebSearch 检索来源的标题超链接格式

对于通过 WebSearch 检索到的来源，必须附上可点击的标题超链接：

**基本格式**：
```
【N】[机构名称]，《[报告标题](URL)》，[年份]年。
```

**示例**：
```
【3】国家能源局，《[中国天然气发展报告2025](https://www.nea.gov.cn/2025-08/15/c_xxxxxx.htm)》，2025年。
【5】国际能源署（IEA），《[World Energy Outlook 2025](https://www.iea.org/reports/world-energy-outlook-2025)》，2025年。
【7】国家统计局，《[2025年国民经济和社会发展统计公报](https://www.stats.gov.cn/sj/zxfb/202602/t20260228_xxxxxx.html)》，2025年。
```

**规则**：
- 使用 Markdown 超链接语法 `[标题](URL)`
- URL 必须完整、可访问
- 知识库来源如有对应的原始发布网址，也应附加超链接

## 附表A：待补充引用清单

当报告中存在 `[待补充引用]` 标记的断言时，在参考文献汇总后附上：

```
## 待补充引用清单

| 优先级 | 原编号 | 缺失事实 | 对报告结论的影响 | 所需资料 | 建议获取渠道 |
|---|---|---|---|---|---|
| P0 | — | 集团2025年管道总里程 | 直接影响规模判断和投资需求测算 | 集团管道资产台账 | 集团生产经营部/规划部 |
| P1 | — | XX省天然气消费增长率 | 影响区域市场判断 | 省能源局统计数据 | 省能源局官网/国家能源局 |
| P2 | — | 同行业数字化投入对标 | 影响保障措施可信度 | 行业数字化投入调研 | 行业协会/研究机构 |
```

### 优先级定义

| 优先级 | 定义 |
|---|---|
| P0 | 核心结论依赖此数据，缺失将导致结论不可信 |
| P1 | 影响分项判断的可信度和精确度 |
| P2 | 影响论据的充实程度，但不影响核心结论 |

## 汇总交付要求

1. 参考文献汇总与标注后的报告正文同时交付。
2. 汇总位于报告正文末尾，作为独立章节「参考文献」。
3. 待补充引用清单紧跟在参考文献之后。
4. 每条汇总记录均可从正文中的 【N】 标注快速定位。
5. WebSearch 检索来源附有可点击的标题超链接。
6. 交付前检查：汇总记录数 ≥ 正文中使用的最大编号（考虑同一来源多处引用的情况）。
