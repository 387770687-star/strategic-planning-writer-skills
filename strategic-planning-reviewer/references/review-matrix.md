# Review Matrix

Score each dimension from 1 to 5.

| Dimension | Weight | 5-Point Standard | Common Failure |
|---|---:|---|---|
| Strategic main line | 15% | clear, differentiated, decision-ready | generic planning language |
| Structure | 15% | complete, balanced, no overlap | missing chapters, repeated content |
| Logic closure | 20% | situation-objective-task-safeguard-risk loop | analysis and tasks disconnected |
| Evidence | 20% | current, sourced, traceable | invented or unsourced data |
| National PipeChina fit | 10% | reflects platform role and pipeline reform | generic energy company narrative |
| Operability | 10% | tasks have path, sequence, resources | vague slogans |
| Language | 10% | formal, concise, SOE style | casual or inflated wording |

## Red Flags

- Exact current company data without source.
- Strategy claims inconsistent with National PipeChina platform role.
- Objectives with no baseline or implementation path.
- Risks not addressed by safeguards.
- New energy/strategic emerging businesses described as mature without evidence.
- Government support requests written as demands rather than coordination needs.
- "全国领先、世界一流、历史性突破" used without official basis.

## Character Count Review

For full reports:

- below 18,000: insufficient for requested long-form report
- 18,000-20,000: close but not enough if user requested over 20,000
- 20,000-24,000: acceptable
- 24,000-32,000: strong delivery range
- above 35,000: check for repetition and appendix split
