# Correction Playbook

Use this to convert review findings into fixes.

## If the Main Line Is Weak

Add a strategic main line paragraph:

`本规划以...为统领，以...为主线，以...为重点，统筹...和...，推动...，为...提供支撑。`

Then make every chapter connect back to that main line.

## If Situation Analysis Is Policy Dumping

For each policy paragraph, add:

1. policy requirement
2. impact on oil and gas infrastructure
3. implication for National PipeChina
4. planning response

Delete copied policy text that does not support a planning conclusion.

## If Objectives Are Empty

Rewrite objectives into:

- scale target
- capability target
- efficiency target
- safety target
- digital target
- value target
- stage target

Use placeholders for missing numbers.

## If Deployment Is Vague

For each task, add:

- action object
- region/business scope
- sequence
- resource or mechanism
- expected result

Weak: "持续推进智慧管网建设。"

Better: "围绕统一调度、完整性管理、智能巡检和应急指挥四类高价值场景，分阶段推进数据治理、模型建设和业务系统贯通，形成覆盖规划、建设、运营、维护、客服的智慧管网能力。"

## If Risks Are Generic

For each risk, specify:

- trigger
- affected business
- impact path
- warning indicator
- mitigation mechanism

## If Report Is Under 20,000 Characters

Expand by adding:

- executive summary
- SWOT cross-strategy section
- scenario analysis
- indicator system table and explanatory text
- business-line deployment subsections
- investment and value assessment
- risk matrix
- implementation schedule

Do not expand by repeating background.
